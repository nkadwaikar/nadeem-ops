# Templates

This directory contains templates for creating lab documents.

## Available Templates

### lab-template.md
A comprehensive template for creating lab documents. Includes sections for:
- Overview and objectives
- Prerequisites
- Step-by-step instructions
- Verification steps
- Troubleshooting
- Summary and next steps

## Using Templates

1. Copy the appropriate template
2. Rename it to match your lab topic
3. Fill in all sections with your content
4. Save it to the appropriate `docs/` subdirectory
5. Update the main README with a link to your document

## Template Guidelines

- Keep instructions clear and concise
- Include code examples with expected outputs
- Add troubleshooting tips for common issues
- Provide verification steps
- Include prerequisites and estimated time
- Update the metadata (author, date, difficulty) at the bottom
